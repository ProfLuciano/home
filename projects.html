WebAnalytics

The purpose of this project is to collect data from entities on the Web and perform data analysis on them. These are the analyses we provide so far:

Airfare Analysis

Data analysis on promotional airfares of domestic and international flights (103,438 trips) departing from Brazilian cities collected from the website MelhoresDestinos.

Try out our Demo: https://airfare.shinyapps.io/sales/

RealEstate Analysis

Analysis of real estate data from thousands of Brazilian real-estate websites.

KB Enhancement

In this project we aim to organize, augment and link entities in knowledge bases using unstructured data.
